iShare - векторные социальные кнопки "поделиться" в соцсетях.

== Description ==

iShare Social Share Buttons - бесплатный русскоязычный плагин кнопок "поделиться" в соцсетях


= Для чего нужен плагин iShare? =

Плагин позволяет установить на сайте красивые векторные кнопки "поделиться" в соцсетях, которые настраиваются в удобном конструкторе под любой дизайн сайта.
Кнопки увеличат посещаемость вашего сайта до 25% ( подробнее тут - http://share.itraffic.su/ )


= iShare поддерживает социальные сети и мессенджеры: =

* Более 25 популярных социальных сетей: Вконтакте, Facebook, Одноклассники, Twitter, Google+ и т.д.
* Более 3 популярных мессенджеров: Viber, WhatsApp, Telegram и др.
* Мы постоянно добавляем новые кнопки соцсетей.


= Уникальность сервиса iShare: =

* **Векторные кнопки**
  Мы не используем ни 1 картинки и все изображения кнопок и их формы создаются наши кодом "на лету".
* **Уникальные формы**
  Наши формы кнопок тоже векторные. Мы не ограничены только круглыми и квадратными формами и постоянно добавляем новые стильные формы кнопок.
* **Максимальное быстродействие**
  Наш код весит около 40кб и отдается только 1 запросом. В отличие от других сервисов мы не грузим сотни килобайт кода дюжиной запросов к серверу.
* **СЕО-оптимизация**
  Никаких лишних ссылок. Все ссылки выводятся при помощи javascript с атрибутом rel=«nofollow» и безопасны для поисковиков.
* **Безопасность**
  Мы не собираем статистику и др. специнформацию о ваших посетителях и не передаем ее "третьим лицам" в отличие от некоторых аналогичных сервисов.
* **Кнопки "под ключ"** Принимаем заказы на разработку специальных форм кнопок для вашего сайта. Пишите нам на share@itraffic.su


= Полная кастомизация социальных кнопок: =

* более 28 форм кнопок (стилей)
* 3 размера
* эффекты при наведении (увеличение, наклон, поворот по часовой стрелке)
* анимация (пульсация и др. эффекты)
* настройка фона и цвета кнопок
* позиционирование: статическое или фиксированное, вертикальное или горизонтальное, выравнивание слева / справа / по центру
* отображение счетчиков: общий счетчик, счетчики кнопок, счетчики только для соцсетей по которым поделились 

== Installation ==

1. Выберите Плагины -> Добавить новый
1. В строке поиска введите iShare
1. Установите плагин iShare Social Share Buttons и активируйте
1. Настройте внешний вид кнопок в конструкторе и нажмите "Сохранить"
1. Наслаждайтесь!

== Frequently Asked Questions ==


= Правда что ваши кнопки векторные и не используется ни 1 картинки? =

Да, наши кнопки векторные, и создаются кодом "на лету"

= Поддерживает ли ваш виджет социальных кнопок протокол https / ssl? =

Да, код наших кнопок поделиться в соцсети работают корректно как по протоколу http так и по протоколу https.

= Корректно ли работают ссылки, написанные кириллицей с сервисом iShare? =

Да, ссылки, написанные кириллицей, работают с нашим сервисом корректно.

= Будет ли мой сайт работать медленнее после установки ваших кнопок? =

Нет, наш код кнопок не влияет на скорость загрузки страниц вашего сайта т.к. загружается асинхронно всего одним запросом к серверу. Также используется кэширование на стороне клиента (304 Not Modified)

= Можно ли разместить на одной странице несколько панелей с кнопками? =

Да, вы можете устанавливать на одной странице несколько панелей с кнопками.

= Можно ли сделать одинаковые счетчики для разных страниц сайта? =

Можно. Как это сделать читайте здесь http://share.itraffic.su/faq/14#question14


Полный список ответов на часто задаваемые вопросы смотрите на http://share.itraffic.su/faq
